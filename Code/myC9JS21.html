  • Built-in string, math and array methods

  • jQuery & Other JavaScript Libraries

Recommended Reading List

  • JavaScript Step by Step; by Steve Suehring, Microsoft Press -- go beyond this course and at least reinforce basic concepts you learned here
  • JavaScript The Definitive Guide; by David Flanagan, O'Reilly -- full reference book but not beginner friendly, div deeper into implementation details
  • JavaScript: The Good Parts; by Douglas Crockford, O'Reilly -- very thin but really useful implementation book
  • JavaScript Patterns; by Stoyan Stefanov, O'Reilly --Mastery of JavaScript book
  • Learning JQuery; JQuery Reference Guide; both by Chaffer & Swedberg, Packt -- Best book for JQuery as Bob Tabor recommends
