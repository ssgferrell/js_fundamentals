19. Using jQuery to retrieve JSON via AJAX

Get JSON Data

AJAX stands for Asynchronise Javascript And XML, a way to transfer data between opened web page inside user browser and the web server, without having to do full web page reload using HTTP Get and HTTP Post.


JSON is a format of Javascript file, stands for JavaScript Object Notation. Instead of sending AJAX data around, JS programmers prefer to send JSON files around instead


JQuery is used to simplify the process of transfering data
